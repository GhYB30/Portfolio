Ghost Graveyard


Also by Mojito Studios, was recently developed an endless runner for mobile devices with a chilly ambient.
It implements a database to keep the competitive spirit alive! And was also developed in Unity.
I was in charge of implementing the controllers for both the different pop ups and all the audio in the game to create smooth loops.
Aside from that, I also dealt with the lightning, skybox and background to build the thrilling atmosphere and voiced the trailer.

Aventura Gatuna


A bit older and this time completely in Spanish, Aventura Gatuna was developed by McAcos.
It is a 2D, Unity game where I designed half of the characters since we all had a hybrid role.
It takes pride in the successful use of various design patterns such as: state, command, obsever, flyweight, singleton or prototype.

Caída Celestial: El desafío de los exiliados


Again completely in Spanish and developed by McAcos, Caída Celestial is a puzzle-platform game created in Phaser for a university project.
The trailer was once again voices and edited by me and I was fully programming for this one.
It can be played now on itch.io here.
Another version was developed using websockets and an API rest so that it could be played in multiple devices in a same web, keeping track of the users and rankings.
